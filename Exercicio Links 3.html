COMO FUNCIONA A INTERNET?

A internet é baseada em um conceito antigo, conhecido na computação como arquitetura cliente-servidor. Nela, existem vários computadores interligados a uma rede, e esta, interligada a outra, formando assim, um aglomerado de computadores interligados, transferindo e recebendo informações.





























O QUE É UM SERVIDOR WEB?

O servidor web é um software específico que você deve ter instalado no servidor. Ele faz com que o servidor "entenda" as solicitações de páginas recebidas da internet e as envie para outros usuários do mundo todo.





























WORLD WIDE WEB

A World Wide Web (mais conhecida como WWW), significa Teia de Alcance Mundial, é um sistema de documentos em hipermídia que são interligados e executados na internet. Esses documentos podem estar na forma de videos, sons, hipertextos ou figuras. Para visualizar a informação, você deve usar um programa de computador chamado navegador.
